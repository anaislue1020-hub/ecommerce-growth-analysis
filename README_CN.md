# 电商增长分析（E-commerce Growth Analysis）

## 1. 项目概述

本项目基于电商平台交易数据，从用户增长、转化行为与变现能力三个维度，对平台整体增长质量进行系统分析。

分析重点在于判断平台增长是否具有可持续性，并识别驱动用户行为与收入变化的核心因素。

**完整分析报告（Notion）：**
https://abiding-puppet-076.notion.site/E-commerce-Growth-Analysis-User-Behavior-Cohort-Dynamics-and-Monetization-34445a9fe7e28025a52bf1082163c443

---

## 2. 分析目标（业务问题）

本项目围绕以下关键问题展开：

1. 用户规模的增长，是否伴随着转化率的稳定或提升？
2. 用户是否在持续产生复购行为，参与度是否增强？
3. 收入增长主要来自用户规模扩张，还是单用户价值提升？
4. 平台促销活动在用户购买行为中起到了多大作用？

---

## 3. 数据范围与口径说明

- 时间范围：2016 Q3 – 2022 Q2（仅保留完整季度）
- 用户范围：注册年龄（join age）≥ 16
- 订单范围：仅包含成功支付订单（payment_status = 'Success'）
- 队列分析窗口：30天 / 90天

**数据来源（Kaggle）：**
https://www.kaggle.com/datasets/bytadit/transactional-ecommerce

---

## 4. 数据处理与建模

- 对用户表与订单表进行清洗与标准化处理
- 剔除异常年龄与不完整时间数据
- 构建分析用数据表：
  - `customer_analysis`：用户层级特征（年龄、分群、首单时间等）
  - `customer_metrics`：用户聚合行为指标（订单数、收入等）

---

## 5. 核心指标定义

| 指标 | 定义 |
|------|------|
| 转化率（Conversion Rate） | 至少产生1笔订单的用户 / 总用户 |
| 复购率（Repeat Rate） | 至少产生2笔订单的用户 / 已转化用户 |
| 人均订单数（Avg Orders per User） | 总订单数 / 已转化用户 |
| 客单价（AOV） | 总收入 / 总订单数 |
| ARPPU | 总收入 / 已转化用户 |
| 促销订单占比（Promo Order Rate） | 使用促销的订单 / 总订单数 |
| 折扣率（Discount Rate） | 促销金额 /（收入 + 促销金额） |

---

## 6. 分析框架

整体分析分为四个模块：

1. **用户结构与增长趋势**  
   → 分析用户构成及用户规模变化情况  

2. **用户购买行为**  
   → 分析复购情况及用户价值变化  

3. **转化分析（Cohort分析）**  
   → 评估不同时间加入用户的转化表现  

4. **促销行为分析**  
   → 判断促销对购买行为与收入的影响  

---

## 7. 核心发现（Key Insights）

- 平台用户规模持续增长，并在 Q1 和 Q3 存在明显季节性高点  
  → 未观察到稳定的低谷期  

- 不同时间队列的转化率整体呈下降趋势，且在各维度表现一致  
  → 说明转化下滑并非由用户结构变化导致，而可能反映增长质量下降  

- 已转化用户的复购行为有所增强  
  → 留存用户参与度在提升  

- 平台收入增长同时受到用户规模扩张与单用户价值提升的驱动  

- 虽然促销使用较为普遍，但其对整体收入的影响有限  
  → 折扣率长期维持在较低水平  

---

## 8. 业务启示（Business Implications）

- 平台应同时关注“增长规模”与“增长质量”，避免单纯追求新增用户数量  
- 提升新用户早期转化能力，是提升长期用户价值的关键  
- 相比持续拉新，优化留存与复购可能更具性价比  
- 当前促销策略对收入驱动有限，可重新评估其必要性与投入产出比
